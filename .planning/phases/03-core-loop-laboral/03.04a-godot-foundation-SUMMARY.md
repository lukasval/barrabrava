## Decisions Made

- FONT_HEADING 22→20: Phase 3 collapses Heading + Numeric Counter into a single 20px role per UI-SPEC §3b. Phase 1 value was 22; changed with OVERRIDE comment.
- AI display_name null fallback: PibeCard renders "Capo de la Barra #N" client-side when display_name is null or empty. No server change needed.
- RecruitCard set_pick pattern mirrors ClubCard.set_club — same Dictionary-driven setter approach.
- TurnoModal lives in `scenes/TurnoModal.tscn` (root), not `scenes/components/` — per UI-SPEC §5.7 modal classification.
- SkillProgressRing uses Godot 4.3 native Control._draw() + draw_arc() with TAU constant for full-circle arc math.
- complete_tutorial(step, elapsed_ms=0): elapsed_ms only included in JSON payload when > 0, matching server expectation.
