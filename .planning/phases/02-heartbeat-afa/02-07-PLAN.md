**FCMPlugin.java** — Godot 4.3 GodotPlugin subclass with Firebase FCM integration:

```java
package com.barrabrava.fcmplugin;

import android.app.Activity;
import androidx.annotation.NonNull;
import org.godotengine.godot.Godot;
import org.godotengine.godot.plugin.GodotPlugin;
import org.godotengine.godot.plugin.SignalInfo;
import org.godotengine.godot.plugin.UsedByGodot;
import com.google.firebase.messaging.FirebaseMessaging;
import java.util.Arrays;
import java.util.HashSet;
import java.util.Set;

public class FCMPlugin extends GodotPlugin {

    public FCMPlugin(Godot godot) {
        super(godot);
    }

    @NonNull
    @Override
    public String getPluginName() {
        return "FCMPlugin";
    }

    @NonNull
    @Override
    public Set<SignalInfo> getPluginSignals() {
        return new HashSet<>(Arrays.asList(
            new SignalInfo("on_token_received", String.class),
            new SignalInfo("on_message_received", org.godotengine.godot.Dictionary.class)
        ));
    }

    @UsedByGodot
    public void getDeviceToken() {
        FirebaseMessaging.getInstance().getToken()
            .addOnCompleteListener(task -> {
                if (!task.isSuccessful()) {
                    android.util.Log.w("FCMPlugin", "getToken failed", task.getException());
                    return;
                }
                String token = task.getResult();
                // Emit signal back to Godot on main thread.
                runOnUiThread(() -> emitSignal("on_token_received", token));
            });
    }

    @UsedByGodot
    public void subscribeToTopic(String topicName) {
        FirebaseMessaging.getInstance().subscribeToTopic(topicName)
            .addOnCompleteListener(task -> {
                if (!task.isSuccessful()) {
                    android.util.Log.w("FCMPlugin", "subscribeToTopic failed: " + topicName, task.getException());
                }
            });
    }

    @UsedByGodot
    public void unsubscribeFromTopic(String topicName) {
        FirebaseMessaging.getInstance().unsubscribeFromTopic(topicName)
            .addOnCompleteListener(task -> {
                if (!task.isSuccessful()) {
                    android.util.Log.w("FCMPlugin", "unsubscribeFromTopic failed: " + topicName, task.getException());
                }
            });
    }

    // Called by FCMMessagingService to relay messages to Godot.
    void relayMessage(org.godotengine.godot.Dictionary payload) {
        runOnUiThread(() -> emitSignal("on_message_received", payload));
    }
}
```

**FCMMessagingService.java** — FirebaseMessagingService to receive token refreshes + messages:

```java
package com.barrabrava.fcmplugin;

import com.google.firebase.messaging.FirebaseMessagingService;
import com.google.firebase.messaging.RemoteMessage;
import org.godotengine.godot.Dictionary;

public class FCMMessagingService extends FirebaseMessagingService {

    @Override
    public void onNewToken(@NonNull String token) {
        super.onNewToken(token);
        // Token refresh — notify Godot if plugin instance available.
        // In Phase 2 the game will call getDeviceToken() on login; this handles background refresh.
        android.util.Log.i("FCMMessagingService", "Token refreshed (length=" + token.length() + ")");
        // TODO Phase 4: wire to active plugin instance for automatic re-registration.
    }

    @Override
    public void onMessageReceived(@NonNull RemoteMessage message) {
        super.onMessageReceived(message);
        Dictionary payload = new Dictionary();
        if (message.getNotification() != null) {
            payload.put("title", message.getNotification().getTitle());
            payload.put("body", message.getNotification().getBody());
        }
        Dictionary data = new Dictionary();
        for (java.util.Map.Entry<String, String> entry : message.getData().entrySet()) {
            data.put(entry.getKey(), entry.getValue());
        }
        payload.put("data", data);
        android.util.Log.i("FCMMessagingService", "onMessageReceived type=" + message.getData().get("type"));
        // Relay to Godot via plugin instance — best effort (may be null if app is backgrounded).
        // System notification is shown by FCM SDK automatically when app is in background.
    }
}
```

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-2-FCM-04 | Spoofing | onMessageReceived payload | accept | Server-side message validation handles trust; Android app only displays what server sent; no UGC in push payload (D-13 locked copy) |
